Project Brief & Goals

The core objective of the ISS Smartwatch UX challenge.

🛰️
ISS Workspace
Created to assist International Space Station crew members. Replaces laptops or tablets for quick, secondary tasks that require hands-free efficiency.
⚡
Single App Hub
A single dashboard hosting multiple sub-modules. Designed to prevent clutter while ensuring quick notification delivery and status feedback.
🎨
Deep Space UX
High-contrast visual design optimized for dark environments. Pitch black backgrounds (OLED power-saving) combined with distinct status color codes.

App Modules

The four core applications combined within the smartwatch platform.

Timeline Crew Timeline

Displays an agenda view of the daily crew timeline directly on the watch. Astronauts can easily view current and upcoming events or look up past schedules.

  • Scrollable agenda listing all scheduled tasks
  • Day-by-day navigation keys for past and future days
  • Glanceable tiles showing active progress of current tasks

Cautions Caution & Warnings

Color-coded alerts displaying critical station issues. The UI uses high-visibility warnings for urgent events and muted cautions for secondary notices.

  • Glow-coded alert frames indicating level (Warning, Caution, Note)
  • Instant "Acknowledge All" action to dismiss alerts
  • Haptic-coded feedback simulated per alert category

Comms Communication Status

Tracks the space-to-ground link status, indicating whether the vehicle can currently communicate with mission control (voice/video) or is in a zone of exclusion.

  • Dual-ring indicator showing uplink and downlink state
  • Real-time status display of signal latency
  • Voice and video link visual indicators

Timers Procedural Timers

Easily set countdown timers for specific onboard scientific procedures, timeline tasks, or alarms to the next scheduled activity.

  • Circular procedure progress/countdown ring
  • Quick-action control buttons (Start, Pause, Reset)
  • Haptic alerts when timer reaches completion

Deploy to Google Pixel Watch

Follow these steps to run the interactive Wear OS app wrapper on real watch hardware.

Wear OS Deployment Guide
# 1. Open the Wear directory in Android Studio (not the parent folder)
# Close any existing project, then file -> Open -> select "wear/"

# 2. Build the project using gradlew from terminal
cd ~/crew-watch-wireframes/wear
./gradlew :app:assembleDebug

# 3. Enable Wireless Debugging on the Pixel Watch
# Go to: Settings -> System -> Developer options
# Turn ON: ADB Debugging and Debug over Wi-Fi

# 4. Pair device in Android Studio Device Manager (via Wi-Fi)

# 5. Install the compiled APK manually if run fails
~/Library/Android/sdk/platform-tools/adb install -r app/build/outputs/apk/debug/app-debug.apk

# 6. Run on the watch!
